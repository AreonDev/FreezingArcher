Third Party Libraries

We are using a bunch of third party libraries for this framework. Without those perls of code a project like this would not be possible. Many thanks to all the developers creating free libraries and making the source code available and reusable to others.

  • STA.INIFile

    From Moreno Airoldi

    Thank you for this awesome INI file parser. The software is licensed under The Code Project Open License (CPOL)

  • Pencil.Gaming

    From Andy Korth and Antonie Blom

    Pencil.Gaming is an awesome OpenGL wrapper written in C#. It is licensed under MIT. We forked this project to fit our needs.

  • CommandLine

    From Giacomo Stelluti Scala

    Thank you for this awesome command line parsing tool. It is licensed under MIT.

  • Jitter Physics

    From Thorben Linneweber

    A powerful physics engine. The license of this project can be found in the original source code. We created a fork on github.

  • Assimp-Net

    From Nicholas Woodfield

    A powerful parser for 3D model formats. This project is licensed under MIT. We created a fork on github.

  • gwen-dotnet

    From "Omeg"

    A .NET/mono port of the GWEN GUI library. This project is licensed under MIT. We created a fork on github.

  • Webpage creation

  • Jekyll

    From Tom Preston-Werner

    Jekyll is an awesome static webpage generation tool. It is licensed under MIT.

  • Agency Jekyll Theme

    From Rick K. (a.k.a. y7kim)

    Thank for this awesome Jerkyll theme. This theme is licensed under the Apache License.

  • Open Source for the win!
