FreezingArcher

FreezingArcher is a lightweight cross platform framework (currently only GNU/Linux x64 and Windows x64 are officially supported) for 3D game development in C# (mono/.NET). The aim of this project is to offer a lightweight but powerful game development environment in C#. The aim is not to create an easy to use drag 'n drop editor for a bloated all-in-one game engine suitable for every purpose. This framework is from developers for developers.

In its current state the engine is far from being finished. It is under heavy development. This software is free software. It is licensed under the GNU GPLv2. The software is also available for free. So feel free to fork this project and bring some of your ideas back into FreezingArcher via pull requests. You may also test the available components and report bugs or create requests for new features to improve the framework.
